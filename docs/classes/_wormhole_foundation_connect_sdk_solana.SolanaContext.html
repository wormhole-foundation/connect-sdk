      The address of the sender

      The token amount to be sent

    • recipientChain: ChainId | ChainName

      The destination chain name or id

    • recipientAddress: Buffer | Uint8Array

      The associated token address where funds will be sent

    • fromAddress: PublicKeyInitData

      The token account pubkey, owned by fromOwner address

      The id of the token's chain

    • Optional payload: Buffer | Uint8Array

    • Optional commitment: Commitment
